# Структура backend/src

**Точка входа приложения:** `index.js` (Express-сервер, маршруты из `routes/`, БД из `db.js`).

Папки `http/`, `app/`, `config/`, `infra/`, а также `index.ts` — альтернативная/устаревшая реализация на TypeScript; в текущем запуске не используются. Для разработки и деплоя используется только `index.js` и подключаемые им модули.
