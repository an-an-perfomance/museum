# Руководство пользователя — Музей школы №323

Сайт «Музей школы» — это веб-приложение для просмотра фотографий и видео школьного музея КОЦ «Оккервиль» (Школа №323, Санкт-Петербург). Администраторы могут добавлять, редактировать и удалять фотографии и видео, а также управлять пользователями.

## 1. Обзор интерфейса

- **Шапка (header)** — на всех страницах:
  - слева: логотип школы и название «Музей Школы»;
  - по центру: пункты меню **Главная**, **Галерея**, **Видео**; у администраторов дополнительно — **Админ-панель**;
  - справа: кнопка **Войти** или имя пользователя и кнопка **Выйти**.

- **Основная область** — контент текущей страницы (главная, галерея, видео, страница фото или видео, вход, админ-панель).

- **Футер** — внизу страницы: подпись «Музей школы №323 · КОЦ «Оккервиль»» и ссылка **Сайт школы** (переход на основной сайт school323.ru).

## 2. Просмотр контента (все пользователи)

### Главная страница («Главная»)

- Отображается заголовок «Музей школы» и карусель с последними фотографиями (если они есть).
- **Действия:**
  - клик по области страницы — переход в **Галерею**;
  - клик по слайду в карусели — переход на страницу выбранной фотографии.
- Под каруселью — подсказка: «Нажмите в любом месте, чтобы перейти в галерею».

### Галерея

- Сетка карточек с фотографиями: у каждой карточки — превью, название и краткое описание.
- **Действия:**
  - клик по карточке — открывается просмотр изображения в режиме галереи (можно листать другие фото);
  - клик по иконке с подсказкой «Открыть полное описание фото» — переход на страницу этой фотографии с полным описанием.
- Если фотографий больше 50, внизу появляется кнопка **Загрузить ещё** — по нажатию подгружаются следующие фотографии.
- Ссылка «Музей школы №323» в шапке галереи ведёт на главную страницу сайта.

### Раздел «Видео»

- Отображается сетка карточек с видео: превью, название и краткое описание.
- **Действия:**
  - клик по карточке или по кнопке «Смотреть» — переход на страницу выбранного видео;
  - на странице видео доступны воспроизведение, название, полное описание, автор и дата.
- Если видео больше 50, внизу доступна кнопка **Загрузить ещё** для подгрузки следующих записей.
- Хлебные крошки позволяют вернуться в раздел «Видео» или на главную.

### Страница фотографии

- Отображаются: хлебные крошки (Главная → Галерея → название фото), изображение, название, полное описание, при необходимости — автор и дата добавления.
- **Хлебные крошки** — по ссылкам «Главная» и «Галерея» можно вернуться на соответствующие разделы.
- Если фото не найдено — показывается сообщение об ошибке и кнопка **Вернуться в галерею**.

## 3. Вход в систему

- Кнопка **Войти** в правой части шапки открывает страницу входа.
- Необходимо ввести **имя пользователя** и **пароль**, затем нажать **Войти**.
- При успешном входе происходит переход на главную страницу; в шапке отображаются имя пользователя и кнопка **Выйти**.
- Ссылка **Вернуться на главную** на странице входа возвращает на главную без входа.

**Примечание:** Учётные данные выдаёт администратор. Доступ к админ-панели есть только у пользователей с ролью администратора.

## 4. Админ-панель

Доступна только после входа под учётной записью с ролью **ADMIN**. В меню шапки появляется пункт **Админ-панель**.

В админ-панели три вкладки: **Пользователи**, **Фотографии** и **Видео**.

### Вкладка «Пользователи»

- Таблица пользователей: имя пользователя, роль (USER / ADMIN).
- Выбор строк: отметить одного или нескольких пользователей (для части действий требуется выбрать ровно одного).
- **Добавить пользователя** — открывается форма: имя пользователя, пароль, роль. После сохранения пользователь может входить с этими данными.
- **Сбросить пароль** — доступно при выборе **одного** пользователя. Генерируется новый пароль; он показывается в модальном окне. Пароль можно скопировать (кнопка с иконкой копирования) и передать пользователю.
- **Удалить** — удаление выбранных пользователей (подтверждение обязательно).

### Вкладка «Фотографии»

- Список фотографий с превью; при большом количестве — кнопка **Загрузить ещё**.
- **Добавить фото** — форма: название, краткое описание, полное описание (по желанию), выбор файла изображения. После отправки фото появляется в галерее.
- Для каждой фотографии:
  - **Редактировать** — изменить название, краткое и полное описание (без повторной загрузки файла).
  - **Удалить** — удаление с подтверждением.
- Можно отметить несколько фотографий и удалить их одной операцией (**Удалить выбранные**).
- Клик по превью или по кнопке просмотра полного описания открывает полный текст описания в модальном окне.

### Вкладка «Видео»

- Список видео с превью; при большом количестве — кнопка **Загрузить ещё**.
- **Добавить видео** — форма: название, краткое описание, полное описание (по желанию), выбор файла видео. Поддерживаемые форматы: **MP4, WebM, MOV**. Максимальный размер файла задаётся настройками сервера (по умолчанию до 300 МБ). После отправки видео появляется в разделе «Видео» на сайте.
- Для каждого видео:
  - **Редактировать** — изменить название, краткое и полное описание (без повторной загрузки файла).
  - **Удалить** — удаление с подтверждением.
- Можно отметить несколько видео и удалить их одной операцией (**Удалить выбранные**).
- Клик по превью или по кнопке просмотра полного описания открывает полный текст в модальном окне.

## Краткая шпаргалка

| Действие | Где |
|----------|-----|
| Перейти в галерею | Главная — клик по странице или пункт меню «Галерея» |
| Перейти к видео | Пункт меню «Видео» |
| Открыть фото в полноэкранном просмотре | Галерея — клик по карточке |
| Открыть страницу фото с описанием | Галерея — иконка «Открыть полное описание» или клик по слайду на главной |
| Открыть страницу видео | Раздел «Видео» — клик по карточке или кнопка «Смотреть» |
| Войти в систему | Кнопка «Войти» в шапке → страница входа |
| Выйти | Кнопка «Выйти» в шапке (видна после входа) |
| Админ-панель | Пункт меню «Админ-панель» (только для администраторов) |
| Перейти на сайт школы | Ссылка «Сайт школы» в футере |

Если что-то не работает (например, не загружаются фото или видео, не открывается админ-панель), проверьте подключение к интернету и при необходимости обратитесь к администратору сайта.
