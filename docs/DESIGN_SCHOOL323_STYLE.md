# Рекомендации по стилистике в духе school323.ru

На основе анализа [официального сайта КОЦ «Оккервиль», Школа №323](https://school323.ru/) и текущей палитры проекта.

---

## 1. Цвета (уже учтено в проекте)

В `frontend/src/theme/colors.ts` уже задана палитра по логотипу Оккервиля — её стоит сохранять:

| Назначение        | Цвет           | HEX       | Где использовать                          |
|-------------------|----------------|-----------|-------------------------------------------|
| Основной (синий)  | Тёмно-синий    | `#1e3a5f` | Заголовки, ссылки, логотип-текст, кнопки  |
| Акцент (красный)  | Красный        | `#c41e3a` | Важные акценты, «Оккервиль», выделения    |
| Доп. акцент       | Золотистый     | `#c9a227` | Hover ссылок, второстепенные акценты     |
| Фон страницы      | Светло-голубой | `#f0f4fa` | Фон контента, карточек                    |
| Фон блоков        | Белый          | `#ffffff` | Карточки, модалки, детальные страницы    |
| Текст             | Тёмно-серый    | `#2c2c2c` | Основной текст                            |
| Границы           | Светло-серый   | `#e8e8e8` | Разделители, рамки                        |

На основном сайте школы преобладают синий и белый, красный — точечно. Для «Музея школы» текущий баланс (синий основной + красный акцент) уместен и визуально связывает проект с school323.ru.

---

## 2. Типографика

- **Заголовки:** один чёткий шрифт без засечек (как на школьном сайте). В проекте можно оставить системный стек (например, `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto`) или подключить один нейтральный веб-шрифт (например, **PT Sans** или **Open Sans** для кириллицы).
- **Размеры:** крупные заголовки для «Музей школы», средние — для разделов, мелкие — для подписей и вторичного текста.
- **Насыщенность:** заголовки — `600`/`700`, основной текст — `400`, подписи — `400` с приглушённым цветом.

Так сохраняется ощущение официального, но читаемого стиля школы.

---

## 3. Шапка и навигация

- **Логотип** — круглый, слева (как сейчас), рядом название «Музей Школы» в цвете `primary` (#1e3a5f).
- **Меню:** горизонтальное, пункты в том же синем, при наведении — золотистый акцент (`gold`), без пёстрых фонов.
- **Высота шапки:** достаточная для логотипа (уже 88px), без сжатия.

Это визуально приближает шапку к стилю school323.ru.

---

## 4. Контент и карточки

- **Фон страницы:** `backgroundLight` (#f0f4fa).
- **Карточки (галерея, новости):** белый фон, лёгкая тень, граница `border`. Уже используется в галерее — стоит оставить.
- **Кнопки:** основная — `primary` (синий), при необходимости акцентная — `primaryRed` только для важных действий (например, «Удалить», «Сохранить» в админке).

Так сохраняется «официальный», но не тяжёлый вид, как на основном сайте.

---

## 5. Дополнительные рекомендации

- **Версия для слабовидящих:** на school323.ru есть переключатель. Для музея можно позже добавить увеличение шрифта или контрастную тему, не меняя базовую палитру.
- **Ссылки:** цвет `primary`, при hover — `gold`, подчёркивание по желанию (можно только при hover).
- **Футер:** при появлении футера — нейтральный фон (`backgroundLight` или белый), текст `textSecondary`, ссылки в `primary`.
- **Иконки:** стиль Ant Design уже нейтральный; цвет иконок в шапке и кнопках согласовать с `primary`/`gold`.

---

## 6. Чего избегать

- Яркие «мультяшные» цвета, не из палитры логотипа.
- Слишком пёстрые градиенты или агрессивные тени.
- Декоративные шрифты — только читаемые и нейтральные.
